# SecretScanner VSCode Extension

This is the official Visual Studio Code extension for SecretScanner.

## Features

- Automatically scans files when saved.
- Highlights hardcoded secrets, API keys, passwords, and tokens.
- Displays inline diagnostics and alerts right in your editor.

## Requirements

You must have the `secret-scanner` CLI tool installed in your system path.
```bash
pip install secret-scanner-cl
```
